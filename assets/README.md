# 素材目录

本目录包含项目的原始素材文件。

## 目录结构

### `original/`
存放原始素材文件，包括：
- Logo 原始文件
- 图片原始文件
- 视频原始文件

### `documents/`
存放产品相关文档，包括：
- 产品行程文档（Word、PDF 格式）

## 使用说明

- **原始素材**：这些文件是设计稿和原始资源，用于参考和备份
- **已使用的素材**：实际在网站中使用的素材位于 `public/` 目录下
  - Logo: `public/images/logo.png`
  - 产品图片: `public/images/routes/`
  - 视频: `public/videos/hero-background.mp4`
