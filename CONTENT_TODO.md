- [ ] 机器人工坊：孩子们组装机器人

- [ ] 美食体验：点心制作、茶文化

#### 北京
- [ ] 城市地标：故宫、长城、天安门
- [ ] 博物馆：国家博物馆内部
- [ ] AI 体验：科技互动展示
- [ ] 历史文化：孩子们参观历史遗迹

#### 珠海
- [ ] 城市地标：珠海大剧院、港珠澳大桥
- [ ] 航空航天：航空博物馆、飞行模拟器
- [ ] 海洋科普：海洋馆、海滩活动
- [ ] 科技体验：孩子们体验飞行器

## ✅ 优先级排序

### 🔴 **高优先级（立即需要）**
1. Logo 和品牌标识

3. 线路详细信息和价格
4. 联系方式和办公地址
5. 隐私政策和服务条款

## 📌 注意事项

1. **所有文案需要三语言版本**：泰文、英文、中文
2. **图片要求**：
   - 高分辨率（至少 1920px 宽）
   - 提供 WebP 和 JPG 两种格式
   - 确保有使用权限
3. **法律文件**：建议咨询专业律师审核
4. **价格信息**：确保准确无误，包含所有费用说明
5. **联系方式**：确保 7x24 小时可联系
